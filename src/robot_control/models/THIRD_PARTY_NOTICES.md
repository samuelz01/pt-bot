# Third-Party Simulation Assets

This package includes simulation assets from:

- OSRF Gazebo Models, Creative Commons Attribution 3.0 Unported
  (https://creativecommons.org/licenses/by/3.0/).

The active obstacle arena uses the OSRF cardboard box and cinder block assets
copied here. Traffic cones in the active world are simple SDF geometry with
inline materials, not third-party meshes.

The file `worlds/me5413_obstacle_world_full_backup.sdf` is retained only as a
backup of the previous full imported world and is not the launch target.
